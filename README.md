Простой HTML-сайт, показывающий рейтинг популярных русских блюд с рецептами и изображениями.

## Содержание:
- Борщ
- Пельмени
- Блины
- Окрошка
- Солянка

Каждое блюдо имеет:
- Название
- Оценку
- Ссылку на оригинал рецепта
- Изображение

## Как запустить:
1. Открой файл `index.html` в любом браузере.
2. Таблица отобразится автоматически.
